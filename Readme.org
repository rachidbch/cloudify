* CLOUDSTATION PROJECT 
  =warning= This is an unfinished project. Use it with caution

  Cloudstation Project deals with tools installation whereas bash-it is used for configuration 

  Cloudstation Project installs a workable development environment on newly installed ubuntu box.
  This environment should provide tools allowing the user to be fully productive: 

** Tools Install

*** How to install
 $ git clone "https://gitlab.com/mobilefirstcentury/installations.git" ~/cloustation          
 $ source ~/cloudstation/init.sh                                                            # This will test if platform is standard linux or termux
                                                                                            # and will install station.packages or termux.packages
                                                                                            # accordingly
